Clearing Browser Cache

Pc Based Browsers

  • Internet Explorer 7.0
    • From Tools -> Internet Options-> General Tab -> Delete (under "Browsing History")
      -> Delete Files -> Yes -> Ok
  • Internet Explorer 4.x, 5.x and 6.x
    • From Tools -> Internet Options -> General Tab -> Delete Files (under "Temporary Internet Files")
      -> Ok -> Ok
  • Firefox 2.0
    • From Tools -> Clear Private Data -> Cache
  • Firefox 1.5
    • From Tools -> Options -> Privacy -> Cache Tab -> Clear Cache Now -> Ok
  • Firefox 1.0
    • From Tools -> Options -> Clear (next to "Cache") -> Ok
  • Netscape 8
    • From Tools -> Options -> Privacy -> Clear (next to "Cache") -> Ok
  • Netscape 7.1
    • From Edit -> Preferences -> [+] Sign (next to "Advanced") -> Cache -> Clear Cache -> Ok
  • Netscape 4.x, 6.x and 7.0
    • From Edit -> Preferences -> [+] Sign (next to "Advanced") -> Cache -> Clear Memory Cache
      -> Ok -> Clear Disk Cache -> Ok -> Ok
  • Opera
    • From File -> Preferences -> History and Cache -> Empty Now (next to "Disk Cache") -> Ok
  • AOL
    • From Settings -> Internet Properties -> AOL Browser -> Clear Storage (under "Top Speed")

Mac Based Browsers

  • Safari
    • From Safari -> Empty Cache -> Empty
  • Firefox 1.5
    • From Firefox -> Preferences -> Privacy -> Clear Cache Now -> Ok
  • Firefox 1.0
    • From Firefox -> Preferences -> Privacy -> Clear (next to "Cache") -> Ok
  • Netscape 6.x for Mac OS 9.x (earlier)
    • From Edit -> Preferences -> Click the Arrow (next to "Advanced") -> Cache -> Clear Cache -> Ok
  • Netscape 6.x for Max OS X
    • From Netscape/Mozilla -> Preferences -> Click the Arrow (next to "Advanced") -> Cache
      -> Clear Cache -> Ok
  • Netscape 4.x
    • From Edit -> Preferences -> Click the Arrow (next to "Advanced") -> Cache -> Clear Disk Cache
      -> Ok -> Ok
  • Explorer 4.x and 5.x for Mac OS 9.x (earlier)
    • From Edit -> Preferences -> Click the Arrow (next to "Web Browser") -> Advanced
      Empty Now (under "Cache") -> Ok
  • Explorer 4.x and 5.x for Mac OS X
    • From Explorer -> Preferences -> Click the Arrow (next to "Web Browser") -> Advanced
      Empty Now (under "Cache") -> Ok
