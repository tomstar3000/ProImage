Clearing Active X

  • Explorer 7.0
    • Restart Internet Explorer
      From Tools -> Internet Options -> Programs -> Manage Add-ons
      Browse for Image Uploader Control from Aurigma Inc.
      -> Delete
  • Explorer 3.x, 4.x, 5.x and 6.x
    • From your computers Start Menu -> Control Panel -> Add/Remove Programs
      Once the list populates, search for Aurigma Image Uploader
      -> Remove -> Restart Internet Explore
