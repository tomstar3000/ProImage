Clearing Browser Cache

  • Internet Explorer 7.x
    • From Tools -> Internet Options-> General Tab -> Delete (under "Browsing History")
      -> Delete Files -> Yes -> Ok
  • Internet Explorer 4.x, 5.x and 6.x
    • From Tools -> Internet Options -> General Tab -> Delete Files (under "Temporary Internet Files")
      -> Ok -> Ok
  • Explorer 3.x:
    • From View -> Options -> Advanced -> Settings -> Empty Folder
  • Firefox 2.x and Firefox 3.x
    • From Tools -> Clear Private Data -> Cache
  • Firefox 1.5
    • From Tools -> Options -> Privacy -> Cache Tab -> Clear Cache Now -> Ok
  • Firefox 1.0
    • From Tools -> Options -> Clear (next to "Cache") -> Ok
  • Chrome 1.x
    • From Tools -> Delete Private Data -> Details -> Delete
  • Safari 3.x
    • From Safari -> Empty Cache -> Empty
  • Mozilla 1.7
    • From Edit -> Preferences -> Advance -> Cache -> Clear Cache -> Ok
  • Netscape 8
    • From Tools -> Options -> Privacy -> Clear (next to "Cache") -> Ok
  • Netscape 7.1
    • From Edit -> Preferences -> [+] Sign (next to "Advanced") -> Cache -> Clear Cache -> Ok
  • Netscape 4.x, 6.x and 7.0
    • From Edit -> Preferences -> [+] Sign (next to "Advanced") -> Cache -> Clear Memory Cache
      -> Ok -> Clear Disk Cache -> Ok -> Ok
  • Opera
    • From File -> Preferences -> History and Cache -> Empty Now (next to "Disk Cache") -> Ok
  • America Online 4.x:
    • From My AOL -> Preferences -> WWW -> General -> Delete Files
  • America Online 3.x (with Explorer 3.x):
    • From Members -> Preferences -> WWW -> Advanced -> Purge Cache
  • America Online 2.x and Older
    • From Settings -> Internet Properties -> AOL Browser -> Clear Storage (under "Top Speed")
