Clearing Browser Cache

  • Safari
    • From Safari -> Empty Cache -> Empty
  • Firefox 1.5
    • From Firefox -> Preferences -> Privacy -> Clear Cache Now -> Ok
  • Firefox 1.0
    • From Firefox -> Preferences -> Privacy -> Clear (next to "Cache") -> Ok
  • Chrome 1.x
    • From Tools -> Delete Private Data -> Details -> Delete
  • Netscape 6.x for Mac OS 9.x (earlier)
    • From Edit -> Preferences -> Click the Arrow (next to "Advanced") -> Cache -> Clear Cache -> Ok
  • Netscape 6.x for Max OS X
    • From Netscape/Mozilla -> Preferences -> Click the Arrow (next to "Advanced") -> Cache
      -> Clear Cache -> Ok
  • Netscape 4.x
    • From Edit -> Preferences -> Click the Arrow (next to "Advanced") -> Cache -> Clear Disk Cache
      -> Ok -> Ok
  • Explorer 4.x and 5.x for Mac OS 9.x (earlier)
    • From Edit -> Preferences -> Click the Arrow (next to "Web Browser") -> Advanced
      Empty Now (under "Cache") -> Ok
  • Explorer 4.x and 5.x for Mac OS X
    • From Explorer -> Preferences -> Click the Arrow (next to "Web Browser") -> Advanced
      Empty Now (under "Cache") -> Ok
